Local Development Setup

Prerequisites
  • PHP 7.4 or higher
  • MySQL 5.7 or higher
  • Apache/Nginx
  • Composer

Clone Repository

git clone https://github.com/RaiyanRafid/NID-Verification-Site.git
cd NID-Verification-Site

Install Dependencies

composer install

Database Setup

mysql -u root -p
CREATE DATABASE nid_verification;
exit;

Database Configuration 
// config/database.php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
define('DB_NAME', 'your_database');

Submitting Verification Request

  1. Visit the website homepage
  2. Fill in personal details:
    • First Name
    • Middle Name (optional)
    • Last Name
    • Address
    • Phone Number
    • Email Address
  3. Select document type:
    • NID
    • Passport
    • Birth Certificate
  4. Upload document images (front and back)
  5. Submit the form
  6. Save the reference number for tracking

Tracking Verification Status

  1. Visit the tracking page
  2. Enter reference number
  3. View current status

Admin Panel Access

  1. Visit /admin
  2. Login with admin credentials
  3. Default credentials: 
    Username: admin
Password: admin123
Warning: Change default admin credentials immediately after first login!

Managing Verification Requests

  1. View pending requests
  2. Review submitted documents
  3. Verify or reject requests
  4. Add comments if needed
  5. Track verification history

Common Issues

Database Connection Error

  • Check database credentials
  • Verify MySQL service is running
  • Check database user permissions

Upload Issues

  • Check upload directory permissions
  • Verify PHP upload limits in php.ini
  • Check file size and type restrictions

Email Problems

  • Verify SMTP settings
  • Check server firewall settings
  • Test with different email providers
